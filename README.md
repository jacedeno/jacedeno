# Hi there, I'm Jose Cedeno! 👋 

### 💡 About Me
I am a **Senior Reliability Engineer** with 30+ years of experience in heavy industry. I specialize in the convergence of **Operations Technology (OT)** and **Information Technology (IT)**. 

My focus is bridging the gap between physical machinery and data-driven insights through:
* 🏗️ **IIoT Architecture:** Prototyping industrial data pipelines from sensor to dashboard.
* ☸️ **Cloud-Native Infrastructure:** Orchestrating bare-metal Kubernetes (K3s) for high-availability industrial apps.
* 📊 **Predictive Analytics:** Leveraging a Post-Grad in Data Science to turn vibration and sensor data into reliability KPIs.

**My Philosophy:** I don't just monitor assets; I build the infrastructure that makes assets talk.

### 🚀 Recent Featured Project: Purchase Requisition System
A full-stack solution for industrial procurement workflows.
- **Backend:** Python (Flask) with Email Authentication.
- **Database:** PostgreSQL managed via **CloudNativePG** (3-node cluster for high availability).
- **Orchestration:** Deployed as 3 replicas on a bare-metal **K3s cluster**.
- **Connectivity:** Secure access via Cloudflare Tunnels.
